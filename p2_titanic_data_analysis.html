def rearrange_name(str):
    arr = str.split(",")
    if len(arr) == 2:
        return arr[1] + " " + arr[0]
    elif len(arr) == 1:
        return str
   
titanic_data_org["Name"] = titanic_data_org["Name"].apply(rearrange_name)

Did age have a role to play in whether a passenger escaped ? To do this, I assume that if we see a strong correlation between a person age and survival , then we can assume that age has a role to play . However for calculation pearsons R , only rows with that orginally had a non nil age should be considered.
